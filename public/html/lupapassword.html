Ubah Password Anda

Demi keamanan, masukkan password saat ini lalu tetapkan password baru yang memenuhi seluruh ketentuan berikut.

Kriteria Password Baru

Password akan dinyatakan valid saat semua poin di bawah berwarna hijau.

  • Minimal 8 karakter
  • Minimal 1 huruf kapital
  • Minimal 1 huruf kecil
  • Minimal 1 angka
  • Minimal 1 karakter spesial
  • Konfirmasi password harus sama dengan password baru
